# Employee Management System Backend (Node.js/Express)

## Overview
This backend provides RESTful APIs for Employees, Departments, Attendance, Leaves, Payroll, and authentication using JWT.

## Main Features
- REST API for all core resources
- JWT-based authentication
- Modular route/controllers/services structure
- Ready for integration with PostgreSQL/MySQL
